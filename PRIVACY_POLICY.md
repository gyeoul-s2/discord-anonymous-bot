# Privacy Policy (プライバシーポリシー)

## 1. 個人情報の取り扱い
- 本ボットはユーザーの Discord メッセージを匿名化して指定されたチャンネルに送信します。
- ユーザー名や ID はログ用に保存される場合がありますが、第三者に公開されることはありません。
- DM 内容以外の情報（サーバー情報、友達リスト等）は収集しません。

## 2. 保存期間
- ログとして保存される匿名メッセージ情報は、管理者が必要な期間のみ保持します。
- 不要になった情報は適切に削除されます。

## 3. 第三者提供
- 個人情報を第三者に提供することはありません。
- Discord API を通じての基本的な情報の取得は Discord 利用規約に従います。

## 4. 変更について
- プライバシーポリシーは予告なく変更されることがあります。変更後も本ボットを利用することで、変更に同意したものとみなされます。

## 5. お問い合わせ
- プライバシーに関するお問い合わせは [GitHub Issues](https://github.com/gyeoul-s2/discord-anonymous-bot/issues) よりお願いします。
